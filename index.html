Portofolio

Proyek Pernah Saya Buat

Berikut merupakan proyek-proyek yang pernah saya buat, baik itu proyek kuliah ataupun proyek individu.

Proyek UAS Pemrograman Basis Data

Proyek tentang pembuatan aplikasi dekstop menggunakan Netbeans IDE dengan bahasa pemrograman Java dan terintegerasi database Oracle. Pada database juga sudah terdapat relasi antar data master, data transaksi, dan data untuk laporan. Program berisi tentang CRUD penjualan Batik.

Proyek UAS Pemrograman Berorientasi Objek

Proyek tentang pembuatan aplikasi dekstop menggunakan Netbeans IDE dengan bahasa pemrograman Java dan terintegerasi database MySQL. Pada database juga sudah terdapat relasi antar data master, data transaksi, dan data untuk laporan. Program berisi tentang CRUD penjualan Keyboard.

Proyek UAS Kecerdasan Bisnis

Proyek tentang pembuatan jurnal Business Intelligent yang menghasilkan data warehouse, data cleansing, SSIS (SQL Server Integration Services), SSAS (SQL Server Analysis Services), Balanced Scorecard, dan dashboard.

Pembelajaran Tailwind CSS

Pembelajaran tentang penggunaan salah satu framework CSS yang sedang naik daun yaitu Tailwind CSS. Mulai dari instalasi, utility, dark mode, transform & animasi, layouting, responsive, flexbox, grid, dan function & directive.

Proyek UAS Interaksi Manusia dan Komputer

Proyek tentang proses pembuatan UI/UX dari awal dengan tema New Normal. Mulai dari pendahuluan (latar belakang, tujuan, dll.), scenario mapping, sitemap, user flow, user persona, crazy eight, wireframe, dan prototype.

Pembelajaran Pemrograman PHP

Pembelajaran tentang salah satu bahasa pemrograman yang sering digunakan dalam website, terutama pada bidang Backend yaitu PHP. Pembelajaran yang dilakukan mulai dari pemrograman dasar, Object-Oriented Programming, model desain MVC (Model, View, Controller), dan penggunaan framework dengan Laravel.

Pembelajaran Pemrograman Java

Pembelajaran tentang salah satu bahasa pemrograman yang selalu masuk 5 besar dalam survey yang dilakukan Stack Overflow yaitu Java. Pembelajaran yang dilakukan yaitu pemrograman dasar dan Object-Oriented Programming.

Proyek UAS Pemrograman Mobile

Proyek tentang pembuatan aplikasi mobile menggunakan Android Studio dengan bahasa pemrograman Java dan terintegerasi database Firebase. Program berisi tentang CRUD jadwal mata kuliah. Dan terdiri dari 3 bagian halaman penting, yaitu halaman jadwal pada hari terkini; halaman jadwal pada semua hari; halaman about.
